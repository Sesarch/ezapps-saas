## âœ¨ Next.js Doesn't Use "index.html"

- **`page.tsx`** instead of `index.html`
- **`layout.tsx`** for the HTML wrapper
- **Components** that compile to HTML

When you run `npm run dev`, Next.js automatically:
1. Compiles everything
2. Creates HTML pages
3. Serves them at localhost:3000
